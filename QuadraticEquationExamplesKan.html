ಪರಿಹಾರ :ಸಂಖ್ಯೆಯು = x ಆಗಿರಲಿ

x2 + 3x = 28

x2 + 3x – 28 = 0

x2 + 7x - 4x - 28 = 0

x(x + 7) –4 (x + 7) = 0

(x + 7) (x – 4) = 0

x = –7 ಅಥವಾ x = 4

∴ ಬೇಕಾದ ಸಂಖ್ಯೆಯು 4 ಅಥವಾ -7
