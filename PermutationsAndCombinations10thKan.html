ಕ್ರಮಪಲ್ಲಟನೆಗಳು

ಕ್ರಮಪಲ್ಲಟನೆ ವಸ್ತುಗಳ ಒಂದು ಕ್ರಮಬದ್ಧವಾದ ವ್ಯವಸ್ಥೆ.

ಕ್ರಮಪಲ್ಲಟನೆಗಳ ಮತ್ತು ಸಂಯೋಜನೆಗಳು ಪರಿಕಲ್ಪನೆಯನ್ನು ಸಮಸ್ಯೆಗಳು ಮತ್ತು ತಳೀಯ ಎಂಜಿನಿಯರಿಂಗ್ ಪರಿಹರಿಸುವಲ್ಲಿ ಅನ್ವಯಗಳನ್ನು ಹೊಂದಿವೆ.

nPr ರಲ್ಲಿ, n ಮತ್ತು r ಧನಾತ್ಮಕ integers.n> = r ಇವೆ

nPr ಒಂದು ಸಮಯದಲ್ಲಿ 'r' ವಿಷಯಗಳನ್ನು ತೆಗೆದುಕೊಳ್ಳುವ 'n' ವಸ್ತುಗಳ ಕ್ರಮಪಲ್ಲಟನೆಗಳ ಸಂಖ್ಯೆ ಅರ್ಥ.

ಒಂದು ಈವೆಂಟ್ 'm' ವಿವಿಧ ರೀತಿಯಲ್ಲಿ ನಡೆಯುತ್ತದೆ ಮತ್ತು ಇನ್ನೊಂದು ಈವೆಂಟ್ 'n' ರೀತಿಯಲ್ಲಿ ಸ್ವತಂತ್ರವಾಗಿ ಸಂಭವಿಸಿದಲ್ಲಿ ನಂತರ ಒಟ್ಟಿಗೆ ಘಟನೆಗಳು (m * N) ವಿವಿಧ ರೀತಿಯಲ್ಲಿ ಮಾಡಬಹುದು.

ಮೊದಲ 'ಎನ್' ಸತತ ನೈಸರ್ಗಿಕ ಸಂಖ್ಯೆಗಳ ಉತ್ಪನ್ನ N ಮೂಲಕ ಸೂಚಿಸಲಾಗುತ್ತದೆ ಇದೆ! ಮತ್ತು ಅಪವರ್ತನೀಯ n ಮಾಹಿತಿ ಓದಲು ಇದೆ.

    ನೆನಪಿನಲ್ಲಿಡಬೇಕಾದ ವಿಷಯಗಳು

  1. ಕ್ರಮಪಲ್ಲಟನೆ ವಸ್ತುಗಳ ಒಂದು ಕ್ರಮಬದ್ಧವಾದ ವ್ಯವಸ್ಥೆ.

  2. ಕೌಂಟಿಂಗ್ ಮೂಲಭೂತ ಪ್ರಿನ್ಸಿಪಲ್: ಒಂದು ಈವೆಂಟ್ 'n' ವಿವಿಧ ರೀತಿಯಲ್ಲಿ ನಡೆಯುತ್ತದೆ ಮತ್ತು ಇನ್ನೊಂದು ಈವೆಂಟ್ 'n' ರೀತಿಯಲ್ಲಿ ಸ್ವತಂತ್ರವಾಗಿ ಸಂಭವಿಸಿದರೆ, ನಂತರ ಘಟನೆಗಳು ಒಟ್ಟಿಗೆ (m * N) ವಿವಿಧ ರೀತಿಯಲ್ಲಿ ಮಾಡಬಹುದು.

  3. nPr=n(n-1)(n-2)......(n-r+1)

  4. nPr=n!/(n-1)!

  5. nPn=n!

  6. 0!=1

  7. nPo=1

ಸಂಯೋಜನೆಗಳು

ಸಂಯೋಜನೆಯು ವಿವಿಧ ವಸ್ತುಗಳ ಕೇವಲ ಆಯ್ದ.

ಸಾಮಾನ್ಯವಾಗಿ ಒಂದು ಸಮಯದಲ್ಲಿ 'n' ತೆಗೆದುಕೊಂಡು 'r' ವಸ್ತುಗಳ ಸಂಯೋಜನೆಗಳು ಸಂಖ್ಯೆ nCr ಮೂಲಕ ಪ್ರತಿನಿಧಿಸಲಾಗುತ್ತದೆ.

nPr ಮತ್ತು nCr ರಲ್ಲಿ, n ಮತ್ತು r ಎರಡೂ ಧನಾತ್ಮಕ ಪೂರ್ಣಾಂಕಗಳ ಮತ್ತು n>=r

ಬಹುಭುಜಾಕೃತಿ ಬದಿ n ವೇಳೆ, ನಂತರ ಇದು (nC2 = N) ಕರ್ಣಗಳು ಹೊಂದಿದೆ.

    ನೆನಪಿನಲ್ಲಿಡಬೇಕಾದ ವಿಷಯಗಳು

  1. ಸಂಯೋಜನೆಯು ವಿವಿಧ ವಸ್ತುಗಳ ಕೇವಲ ಆಯ್ದ.

  2. ಸಾಮಾನ್ಯವಾಗಿ ಒಂದು ಸಮಯದಲ್ಲಿ 'r' ತೆಗೆದುಕೊಂಡು 'n' ವಸ್ತುಗಳ ಸಂಯೋಜನೆಗಳು ಸಂಖ್ಯೆ nCr ಮೂಲಕ ಪ್ರತಿನಿಧಿಸಲಾಗುತ್ತದೆ.

  4. nCr=n!/(n-1)!r!
